## Vue-cli 使用

##### 安装vue-cli (3.0+)

``` js
    npm install @vue / cli - g
```

##### 查看vue-cli版本

``` js
    vue - V 
``` 

##### 创建vue项目

```
    js
    vue create vue - test / vue ui 使用ui界面创建项目
```

    选择安装 router vuex 

##### 使用vue-element 
```js
    npm install element-ui -S  安装element-ui
```
挂载在vue上
```js
    main.js
    import Element from 'element-ui'     //全局引入element-ui   
    Vue.use(Element)
```
